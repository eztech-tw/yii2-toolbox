安裝
----
這個套件請使用 [composer](http://getcomposer.org/download/) 安裝

執行

```
php composer.phar require --prefer-dist eztech-tw/yii2-toolbox "*"
```

或增加

```
"eztech-tw/yii2-toolbox": "*"
```

到你的 `composer.json` 檔案中
